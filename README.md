# 2FA Email System
OTP system som bruker Gmail SMTP for sending av email.

## Informasjon
- `hover_unfriendly_password` variabelen er bare `password` reference.

> [!IMPORTANT]
> Sender email trenger å være gmail
> 
> Gmail sender må lage/bruke app passord som 'passord'
